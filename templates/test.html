Ton arbre de compétences avec Competenssator

Un projet de création d'arbre de compétences en ligne avec les technos suivantes :
  • Code principal en Python
  • Application Web en Flask avec Jinja2 comme moteur de template
  • YAML pour le fichier de description
  • Les arbres de compétences sont générés en SVG
  • Cette page est stylisée avec Bulma
